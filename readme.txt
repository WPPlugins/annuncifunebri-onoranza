Permette di inserire sul proprio sito gli annunci dell'onoranza funebre che utilizza il servizio annuncifunebri.it.

== Descrizione ==

Con questo plugin è possibile visualizzare sul proprio sito gli annunci dell'onoranza funebre pubblicati sul sito annuncifunebri.it
N.B. Tutte le immagini verranno scaricate dal sito www.annuncifunebri.it e non saranno salvate sul proprio server.

== Installazione ==

1. Carica i file del plugin nella cartella `/wp-content/plugins/annuncifunebri-onoranza`, o installa il plugin attraverso il menu nuovo plugin di WordPress.
1. Attiva il plugin dalla schermata plugin
1. Dal menu Annunci Funebri, inserisci l'#ID dell'onoranza funebre che ti verrà fornito dallo staff di AnnunciFunebri ed eventualmente gli altri parametri di personalizzazione delle pagine utilizzate.
